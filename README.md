# Setup

## Installation

1. .NET: https://dotnet.microsoft.com/en-us/download

## Build

### C#

Run the following commands from the project root:

```commandline
cd csharp
dotnet build
```

## Run

### C#

Run the following commands in the `csharp` directory.

Encrypt a message:

```commandline
dotnet build 1
```

Decrypt a message:

```commandline
dotnet build 2
```
